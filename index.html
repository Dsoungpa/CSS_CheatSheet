CSS Cheatsheet

CSS Font Properties

Tag Example Usecases Description
font-family monospace, serif Sets the font to a specific set of fonts. Priority going from left to right.
font-variant normal | small-caps | inital | inherit Specifies whether or not a text should be displayed in a small-caps font.
font-weight normal | bold | 900 Sets how thick or thin characters in text should be displayed.
font-size 15px | large | 150% Sets the size of a font.
font-style normal | italic | oblique Specifies the font style of a text.
